## How to build

- Environment & Requirement:
    - GCC Version: 13.2.1 20230801
    - CMake 3.27
    - GNU Make: 4.4.1

  ```bash
  build_dir=<build_directory>
  
  cmake -B $(build_dir)
  cmake --build $(build_dir)
  ```

  The built binary files will be located at `$(PROJECT_ROOT)/bin/`.
